Zot Search

The Project

  • Zot Search is built to search for documents in the ics.uci.edu domain

  • The search engine runs on a Flask server and utilized a few Semantic-UI elements for the user facing page

  • Search query dataset on Heroku is limited to the word bank as the free environment is unable to serve the entire dataset

UCI CS 121: Information Retrieval

This was a project course that teaches the introduction to information retreival, took in Winter 2020. The topic includes constructing an inverted index, searching, and ranking retrieved documents by computing cosine scores between query and document vector.
